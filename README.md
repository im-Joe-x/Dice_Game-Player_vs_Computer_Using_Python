# Dice_Game-Player_vs_Computer_Using_Python
A simple command-line Dice Game built with Python where the player competes against the computer using random dice rolls, score tracking, functions, and decision-making logic.

Project Purpose

This mini project was developed to strengthen programming fundamentals by creating an interactive dice game. It provides practical experience in implementing functions, random number generation, loops, data structures, and comparison logic in a real-world gaming scenario.
